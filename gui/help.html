How to Use VidCaptio

Using VidCaptio is simple and straightforward. Here's a step-by-step guide:

  1. Open VidCaptio and create a new project (Project > New project).
  2. Choose the language of the audio in the video.
  3. Click generate captions.
  4. Make any edits you would like in editor.
  5. Click Apply captions.
  6. You can check captioned video by navigating to Current project > Open Captioned video
  7. Save project.
Note: You might not be able to enable captions in some video players. However, you can always use the caption files saved in the project folder.
